Opratori

Tramite gli operatori posso cambiare valori tra di loro

Gli operatori determinano il risultato di un'operazione

Classificazione degli operatori in base al numero di valori che possono combinare: unari, binari, ternario

Operatori aritmetici

Operatori binari berchè combinano due valori (numerici!)
+ addizione
- sottrazione
/ divisione
* moltiplicazione
% modulo o resto

Operatiri aritmetici possono anche essere unari, cioè combinano un solo valore
++ incremento
-- decremento
- negazione -> x=5 -> -x = -5
se ci sono due valori - è sottrazione, se c'è un valore solo - è negazione
-> incremento e decremento li posso applicare solo a delle variabili -> è molto importante la posizione dell'operatore (se è a sinistra o a destra dell'operando)
-> se l'operatore è a sinistra dell'operando l'operazione viene eseguita prima di valutare l'espressione
-> se l'operatore è a destra dell'operando l'operazione viene eseguita dopo di valutare l'espressione

Operatori Logici

Consentono di combinare espressioni booleane (true | false)
&& -> operatore binario -> and
|| -> operatore binario -> or
! -> operatore unario -> not

Operatore ternario

Permette di combinare tre dati tra di loro. Restituisce un valore in base alla valutazione di un'espressione booleana
sintassi: condizione? valore1 : valore2
Verifico se la condizione è vera (true|false) -> se è true restituisce valore1 altrimenti valore2

Operatori relazionari

Utilizzati per confrontare dei valori

OperatoreNome
>Maggiore
<Minore
>=maggiore o uguale
<=minore o uguale
==uguale
!=diverso
===strettamente uguale
!===strettamente diverso

tutte queste espressioni restituiscono un valore booleano
